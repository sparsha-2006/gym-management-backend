## Project Overview
This project is a secure and scalable Gym Membership Management Backend System developed using Node.js, Express.js, MongoDB, and JWT Authentication.

The system supports:
- User Authentication
- Role-Based Access Control
- Membership Management
- Attendance Tracking
- Workout Management
- Admin Operations
  
## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs
- Express Validator
- 
## Features

### Authentication
- User Registration
- User Login
- Password Hashing using bcrypt
- JWT Token Authentication
### Roles
- Admin
- Trainer
- Member
### Membership Module
- Subscribe Membership Plans
- Membership Details
- Membership Expiry Handling
### Attendance Module
- Check-In
- Check-Out
- Attendance History
- Duplicate Attendance Prevention
### Workout Module
- Create Workout Plans
- Assign Exercises
- Difficulty Levels
### Admin Module
- View Users
- Block/Unblock Users
- Analytics Dashboard
### Security Features
- Protected Routes
- Role-Based Authorization
- Input Validation
- Centralized Error Handling
